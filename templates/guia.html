Guía de uso del sistema

Aprende cómo utilizar el repositorio institucional.

1. Crear una cuenta

Para utilizar todas las funciones del sistema, primero debes registrarte mediante el botón “Crear cuenta”.

2. Iniciar sesión

Una vez registrado, inicia sesión con tu usuario y contraseña para acceder al repositorio.

3. Subir proyectos

Los usuarios pueden subir proyectos académicos en formato PDF agregando información como:

  • Título
  • Descripción
  • Carrera
  • Autor

4. Buscar proyectos

El sistema permite buscar proyectos por:

  • Categoría
  • Carrera
  • Autor
  • Palabras clave

5. Descargar documentos

Los usuarios pueden visualizar y descargar archivos PDF disponibles en el repositorio.
